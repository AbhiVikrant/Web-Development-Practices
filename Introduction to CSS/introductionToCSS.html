Adding Color to your page::

There are different ways of adding color like simply writting color name or providing rgb or rgba value. We will discuss it in detail here

We are creating four div and adding background color to them using four different methods

  • Adding color using color name itself in style as color: red;

  • Adding color using rgb value which means providing red, green blue value which goes from 0 to 255. RGB value for black is 0 0 0 and white is 255 255 255. For red it is 255 0 0 and blue as rgb(0,0,255).

  • Adding color using rgba which stands for red, green,blue,alpha. Alpha is here for obacity factor which range from 0 to 1.

  • Adding color using hex code which represents the hexadecimal format of rgba.

  • Adding color using HSL or HSLA which means Hue,Saturation,Lighness and Alpha factor. Hue goes from 0 to 255 as hue for black is 0, green is 120.

Introduction to CSS Unit

CSS unit is mainly divided into two parts as absolute and relative. Absolute means have specific value like inch,cm,meter in daily use unit. In CSS, pixel written as px has same concept and 1px is absolute to 1/96 inch. But it is not every time equal to 1 px of device pixel. It may be less than one pixel of device unit.

Relative Unit: CSS has some relative unit as percentage. Let us assume a div as parent having height as 200px and width as 300px. We can create an inner div inside this div having exact half the height and width of parent div by giving pixel for height as 100px and pixel for width as 150px but this is fixed, if we increase the height of parent div then child div will not adjust it according to parent div. In this case we use relative css unit and if we provide 50% to both height and width it will adjust according to parent.Examples as:

Another relative unit is rem which means that it takes the unit as font-size relative of the root element that is the default font size of html. where as em takes the font size of parent.

Another unit is vw and vh as viewwidth and viewheight for screen size.

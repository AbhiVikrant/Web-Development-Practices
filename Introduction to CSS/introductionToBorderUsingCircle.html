Introduction to Border

In CSS, we can provide border to an element using creating style as border-width or border having width type and color. We can also set border of different width across all side as border-width: 2px 4px 6px 8px which is for top right bottom and left. If we write as border-width: 2px 4px 8px it means top as 2px left and right as 4px and bottom as 8px. We can also specify specific border as border-left or border-right or top or bottom. Types of border as dotted, solid , doubled, inset,unset or outset. The default value of border-style as unset and it can be changed to any to show border.
